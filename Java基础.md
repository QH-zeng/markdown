# 集合Collection

* **集合**：集合是java中提供的一种容器，可以用来存储多个数据。

集合与数组的区别：

* 数组的长度是固定的（数组一旦定义了，就不能再修改数组的长度）。

  集合的长度是可变的（可以不断往集合添加元素）。

* 数组中存储的是同一类型的元素，可以存储任意类型数据。

  集合存储的都是引用数据类型。如果想存储基本类型数据需要存储对应的包装类型。（String Integer...）
